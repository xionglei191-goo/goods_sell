## 8. 组件规范

### 数据表格
- 支持列排序、筛选
- 行悬浮高亮
- 支持分页(默认20条/页)
- 操作按钮在最右列
- 移动端转为卡片列表

### 表单
- 标签在上方(Label on top)
- 必填字段带红色星号
- 实时验证+错误提示
- 操作按钮固定在底部

### 按钮层级
- Primary: 实心主色按钮（页面主操作）
- Secondary: 描边按钮（次要操作）
- Ghost: 文字按钮（更多操作）
- Danger: 红色按钮（删除/取消）

### 数据卡片(Dashboard)
- 白色卡片+轻微阴影
- 图标+数字+趋势箭头
- Hover时微上移动画

## 9. 动效规范

```css
--transition-fast: 150ms ease;    /* 按钮hover */
--transition-base: 200ms ease;    /* 一般过渡 */
--transition-slow: 300ms ease;    /* 弹窗/抽屉 */
--transition-spring: 300ms cubic-bezier(0.34, 1.56, 0.64, 1); /* 弹性效果 */
```
