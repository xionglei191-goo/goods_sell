# 华启酒饮数字渠道平台 — 功能模块设计

## 模块一：进销存管理

### 1.1 产品管理
**页面清单：**
| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 产品列表 | /products | 表格展示，搜索/筛选/排序，导入导出 |
| 新增产品 | /products/new | 表单：基础信息+价格+分类+品牌+图片+分单阈值 |
| 编辑产品 | /products/[id]/edit | 修改产品信息 |
| 产品详情 | /products/[id] | 查看详情+库存记录+销售历史 |
| 分类管理 | /products/categories | 树形分类管理(三级) |
| 品牌管理 | /products/brands | 品牌CRUD |

### 1.2 库存管理
| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 库存总览 | /inventory | 当前库存列表，库存预警，库存价值 |
| 入库操作 | /inventory/stock-in | 选产品→输数量→选类型(采购/退货/调整) |
| 出库操作 | /inventory/stock-out | 选产品→输数量→关联订单 |
| 库存盘点 | /inventory/check | 创建盘点→录入实际→自动算差异→确认 |
| 变动记录 | /inventory/records | 所有出入库流水记录 |

### 1.3 采购管理
| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 采购订单 | /purchase | 采购单列表+创建 |
| 供应商管理 | /purchase/suppliers | 供应商CRUD |

---

## 模块二：销售管理

### 2.1 订单管理
| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 订单列表 | /orders | 全部订单，多维筛选(状态/日期/客户/类型) |
| 订单详情 | /orders/[id] | 订单信息+商品明细+分单记录（含综合评分原因）+物流+支付 |
| 手动开单 | /orders/new | 管理员/销售员手动创建订单(线下场景) |

### 2.2 客户管理
| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 客户列表 | /customers | 客户表格，按类型/销售员/标签/动态分层筛选，展示高价值团购、餐饮店、烟酒店、宴席客户、普通消费者 |
| 客户详情 | /customers/[id] | 档案+订单历史+欠款+画像+拜访记录，展示客户分层、画像标签和下一步维护建议 |
| 拜访记录 | /sales/visits | 销售员拜访日志 |

### 2.3 经销商管理
| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 经销商列表 | /dealers | 经销商列表、动态分层、分层筛选、接单/库存/冲突/推广指标 |
| 经销商详情 | /dealers/[id] | 门店信息+接单记录+结算+绑定业务员 |
| 经销商政策 | /dealers/[id]/policy | 服务半径、接单金额、可接品牌、价格权限、拒单规则 |
| 地图总览 | /map | 高德地图+经销商标注+订单热力图 |

### 2.4 销售报表
| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 销售报表 | /sales | 按时间/产品/客户/销售员的销售统计图表 |
| 业绩排名 | /sales/ranking | 销售员业绩排行榜 |
| 业务员绩效 | /salespeople | 地推经销商、绑定客户、线索转化、报价转化、推广码和复购表现看板 |

## 模块三：财务对账

| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 财务总览 | /finance | 应收/应付/收入总览+趋势图 |
| 应收账款 | /finance/receivable | 按客户的欠款列表+账龄分析 |
| 收款登记 | /finance/payments | 登记回款，核销订单 |
| 应付账款 | /finance/payable | 向供应商的未付款 |
| 对账单 | /finance/statements | 按客户/时间段生成对账单，导出PDF/Excel |
| 利润分析 | /finance/profit | 毛利报表(按产品/客户/时段) |

## 模块四：客户入口与场景化订货

| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 平台首页 | /shop | 区域供货定位+渠道保护说明+四个场景入口 |

| 产品目录 | /shop/catalog | 分类浏览+搜索+筛选+排序 |
| 产品详情 | /shop/product/[id] | 图片轮播+规格+价格+加购+评价 |
| 购物车 | /shop/cart | 商品列表+数量+小计+合计 |
| 结算 | /shop/checkout | 地址选择+支付方式+确认下单 |
| 我的订单 | /shop/my-orders | 订单列表+状态+详情+取消/退款 |
| 账户中心 | /shop/account | 个人信息+地址管理+优惠券+积分 |
| AI选品互动 | /shop/ai-chat | 宴席配酒、企业团购/送礼、门店补货、新品试饮、知识库问答，自动生成线索/询价建议 |
| 趣味互动 | /shop/fun | 节气养生+AI测试+签到+新品试饮资格 |

## 模块六：线索、询价与渠道经营

| 页面 | 路由 | 功能描述 |
|------|------|---------|
| 线索池 | /dashboard/leads | 展示 AI 互动、扫码、询价、注册沉淀的客户线索 |
| 询价单 | /dashboard/inquiries | 宴席/团购/门店补货询价，支持分派业务员和生成报价 |
| 报价单 | /dashboard/quotes | 报价方案、商品组合、有效期、转订单记录 |
| 推广码管理 | /dashboard/promoters | 业务员/经销商推广码、来源、扫码与转化统计 |
| 地推试点 | /dashboard/channel-pilot | 为业务员批量绑定试点经销商，并生成业务员地推码、经销商门店码 |
| 新品推送 | /dashboard/product-pushes | 按画像/客户分层筛选客户，生成场景话术，跟踪打开、咨询、试饮、下单、复购并输出复盘建议 |
| 渠道冲突 | /dashboard/channel-conflicts | 跨区、低价、拒单、库存、投诉异常台账，支持责任人分派、状态处理和关闭记录 |

## 模块七：仪表盘(Dashboard)

管理员首页展示：
- 今日数据卡片：订单数、销售额、新增客户、待处理事项
- 销售趋势图（近7天/30天）
- 订单状态分布（饼图）
- 库存预警列表（低库存TOP10）
- 应收账款TOP10（欠款最多的客户）
- 销售员业绩排行
- 最近订单列表
- 线索数、询价数、报价转化率、经销商响应时长、新品推送转化
- 公司经营看板：区域经营、经销商分层、客户类型、新品推送复盘、业务员地推效果和渠道冲突趋势
