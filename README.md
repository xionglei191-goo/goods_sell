# 华启商城 (HuaQi Mall)

> 湘潭区域批发分销 + 线上零售一体化平台

## 项目简介

华启商城是一套面向区域批发分销企业的全链路数字化平台，涵盖**进销存管理**、**销售管理**、**财务对账**和**线上商城**四大核心模块。系统以"线上商城 + 智能分单 + AI营销"为核心，实现从获客、下单、履约到回款的全流程数字化。

## 业务模式

```
品牌厂商 → 区域总代理(华启) → 线上商城(C端散客) + 线下经销商(B端)
                                    ↓
                              智能分单引擎
                            ↙            ↘
                    散单→就近经销商     大单→总仓直发
```

## 核心特色

- 🛒 **线上商城**：面向湘潭市消费者的 B2C 商城，支持在线支付
- 🗺️ **智能分单**：基于地图的订单智能路由，散单分配就近经销商
- 🤖 **AI 客服**：大模型驱动的智能客服 + 用户画像 + 精准营销
- 📊 **数据驱动**：多维度报表，实时掌握经营状况
- 📱 **移动优先**：响应式设计，适配手机/平板/PC

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Next.js 15 (App Router) |
| 语言 | TypeScript |
| UI | shadcn/ui + Tailwind CSS 4 |
| 数据库 | PostgreSQL + Prisma ORM |
| 认证 | NextAuth.js (Auth.js) |
| 地图 | 高德地图 API |
| AI | 统一大模型网关（Anthropic 兼容）/ DashScope / DeepSeek |
| 支付 | 微信支付 |

## 快速开始

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env

# 初始化数据库
npx prisma migrate dev

# 填充演示数据
npx prisma db seed

# 启动开发服务器
npm run dev
```
