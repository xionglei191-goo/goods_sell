# 华启酒饮数字渠道平台 (HuaQi Channel)

> 湘潭区域酒水饮料总代的数字化获客、渠道保护与经销商协同平台

## 项目简介

华启酒饮数字渠道平台是一套面向区域批发分销企业的经营系统，涵盖**客户接入/询价**、**渠道保护型分单**、**经销商协同**、**进销存管理**、**销售财务**和**AI画像营销**。系统不以绕开经销商做传统 B2C 电商为目标，而是帮助区域总代把业务员地推、下游经销商私域、终端客户需求和新品推广沉淀到统一平台。

## 业务模式

```
品牌厂商 → 区域总代理(华启) → 业务员地推/经销商私域/客户场景入口
                                    ↓
                         线索、询价、下单、AI互动画像
                                    ↓
                              渠道保护分单引擎
                            ↙            ↘
                    小单→就近经销商     大单/团购→公司/业务员报价
```

## 核心特色

- 🛒 **客户下单与询价入口**：面向宴席、企业团购、门店补货和普通散客的场景化入口
- 🗺️ **渠道保护型分单**：小单优先就近经销商履约，大单/团购进入公司或业务员报价流程
- 🤖 **AI 选品互动**：宴席配酒、送礼推荐、门店补货等互动采集需求并生成画像
- 🤝 **业务员/经销商协同**：推广码、门店链接、客户归属、接单/拒单/库存上报闭环
- 📊 **渠道经营数据**：线索、询价、分单、经销商活跃、新品动销和复购转化看板
- 📱 **移动优先**：响应式设计，适配手机/平板/PC

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Next.js 15 (App Router) |
| 语言 | TypeScript |
| UI | shadcn/ui + Tailwind CSS 4 |
| 数据库 | PostgreSQL + Prisma ORM |
| 认证 | NextAuth.js (Auth.js) |
| 地图 | 高德地图 API |
| AI | 统一大模型网关（Anthropic 兼容）/ DashScope / DeepSeek |
| 支付 | 微信支付 |

## 快速开始

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env

# 初始化数据库
npx prisma migrate dev

# 填充演示数据
npx prisma db seed

# 启动开发服务器
npm run dev
```
